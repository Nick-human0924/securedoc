隐私说明

核心原则：零云端 · 零日志 · 零残留

SecureDoc 是一款完全在浏览器本地运行的文档处理工具。你的文件从始至终不会上传到任何服务器。

数据处理

  • 本地处理：所有 PDF 操作（合并、拆分、压缩、涂黑）都在你的浏览器中使用 JavaScript 完成
  • 不上传：文件内容不会发送到我们的服务器或任何第三方服务
  • 不存储：我们不保存任何用户数据、文件信息或处理记录
  • 自动清空：刷新页面或关闭标签后，所有临时数据自动清除

技术实现

  • 使用 pdf-lib.js 在浏览器端直接处理 PDF 文件
  • 通过 Blob URL 生成本地下载链接，不涉及后端传输
  • 纯静态页面部署在 GitHub Pages，无服务器端代码
