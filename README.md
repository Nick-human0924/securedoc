# SecureDoc - 本地隐私文档处理工具

🔒 主打"本地处理、隐私保护"的网页端文档工具

## 核心特性

- ✅ **本地处理** - 所有文档处理在浏览器本地完成
- ✅ **隐私保护** - 文档内容不上传服务器
- ✅ **不留痕迹** - 关闭页面后自动清空所有内容
- ✅ **即开即用** - 无需注册登录

## 本地使用

直接在浏览器中打开 `index.html` 即可使用，无需服务器。

## 隐私说明

所有文档处理均在您的浏览器中通过 JavaScript 直接完成：
- 文档内容**不会**上传到服务器
- 处理过程在您的设备本地完成
- 关闭页面后自动清空所有数据
