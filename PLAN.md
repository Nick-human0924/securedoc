# 本地隐私文档处理工具 - 开发计划

## 技术选型
- **前端框架**: 原生 HTML/CSS/JavaScript（轻量、无构建依赖）
- **PDF处理**: pdf-lib.js（纯前端，浏览器端处理PDF）
- **Word处理**: mammoth.js（Word转HTML再处理）
- **部署**: GitHub Pages
